# wordTopdf  
A little wordTopdf demo, It's written in Vue.js, element-ui.js and Node.

## Requirements
node 8  
vue-cli 3.0

## Installation
```
$ npm install
$ cd server
$ npm install
```


## Developers
```
npm run serve
cd server
npm start
```
Open http://localhost:8080/
